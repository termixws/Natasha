# Аренда квартир - Frontend

Фронтенд-приложение для сервиса аренды квартир на React + TypeScript + Vite.

## Технологии

- React 18
- TypeScript
- Vite
- TailwindCSS
- React Router DOM
- React Query
- React Hook Form
- MSW (Mock Service Worker)

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Сборка для продакшена

```bash
npm run build
```

## Функциональность

### Авторизация
- Страница входа: `/login`
- Страница регистрации: `/register`
- Тестовый пользователь: `user@example.com` / `password123`

### Основные страницы
- **Главная** (`/`) - список квартир с фильтрами и поиском
- **Детали квартиры** (`/apartments/:id`) - подробная информация о квартире
- **Бронирование** (`/book/:id`) - форма создания бронирования (требует авторизации)
- **Мои бронирования** (`/bookings`) - список активных и прошедших броней (требует авторизации)

### Фильтры
- Поиск по названию
- Город
- Диапазон цен
- Количество комнат
- Пагинация (10 квартир на страницу)

## API Mock

Все API запросы замоканы с помощью MSW (Mock Service Worker). Реальный бэкенд не требуется.

### Endpoints

- `POST /api/login` - вход
- `POST /api/register` - регистрация
- `GET /api/me` - текущий пользователь
- `GET /api/apartments` - список квартир
- `GET /api/apartments/:id` - детали квартиры
- `POST /api/bookings` - создать бронирование
- `GET /api/bookings` - список бронирований

## Структура проекта

```
src/
├── api/            # API клиенты
├── components/     # React компоненты
├── pages/          # Страницы приложения
├── context/        # React контексты (Auth)
├── hooks/          # Кастомные хуки
├── mocks/          # MSW моки и данные
├── types/          # TypeScript типы
├── router/         # Конфигурация роутинга
└── styles/         # Глобальные стили
```
